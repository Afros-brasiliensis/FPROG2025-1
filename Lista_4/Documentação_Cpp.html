📘 Introdução ao C++

C++ é uma linguagem compilada, poderosa e orientada a objetos. Muito usada em sistemas, jogos e aplicações críticas.

📌 Instalação no Windows

Use o MinGW. Instale de https://sourceforge.net/projects/mingw/.

g++ arquivo.cpp -o programa

📌 Instalação no Linux

sudo apt update
sudo apt install g++

📌 Instalação no Mac

xcode-select --install

🧱 Estrutura Básica

#include <iostream>
using namespace std;

int main() {
    cout << "Olá, mundo!" << endl;
    return 0;
}

🔧 Comandos Básicos

int a = 10;
float b = 3.14;
if (a > 5) {
    cout << "Maior que 5";
}

🧰 Bibliotecas Comuns

  • #include <iostream> — Entrada e saída
  • #include <vector> — Vetores dinâmicos
  • #include <map> — Dicionários

🔌 Integrações com outras linguagens

Exemplo com Python (pybind11):

#include <pybind11/pybind11.h>
int soma(int a, int b) { return a + b; }
PYBIND11_MODULE(modulo, m) {
  m.def("soma", &soma);
}

🚀 Dicas e Boas Práticas

  • Use #pragma once em headers
  • Prefira std::vector a arrays
  • Evite using namespace std; em headers
